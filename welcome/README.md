# Welcome app

## Creating a venv and installing Flask

### Make a venv

```terminal
python -m venv venv
```

Or

```terminal
python3 -m venv venv
```

### Activate the venv

```terminal
source venv/bin/activate
```

### Deactivate the venv

(When done with)

```terminal
deactivate
```

### Using `pip`

List what is installed:

```terminal
pip list
```

Installing Flask using `pip`:

```terminal
pip install flask
```

## Creating your first simple text based Flask app

### Running the app

```terminal
python app.py
```
